Creating a new component

  1. navigate to the folder you want to create the component in eg; a subcomponent of the panel go to src/app/panel/
  2. type ng g c COMPONENT_NAME
  3. go to src/app/test/test.component.html and paste <app-COMPONENT_NAME> </app-COMPONENT_NAME>
  4. to view component go to localhost:4200/test in the browser
